Resume Scanner-LTTS

Why

Resume tracking is a necessary and critical component of an effective recruiting and hiring process. Hiring management needs an applicant tracking system to streamline the flow of incoming resumes and to manage the process of screening and evaluating applicants.

How

This applicatoin uses recommendation engine techniques such as KNN, content based filtering for matching job description with multiple resumes. It uses ML to provide scores for each resume ranked.







How to Use

The admin can upload new resumes and also update the Job Description. Users can select a particular vertical and search for the desired Job Description file to rank the resumes and also recieve scores for each resume .
